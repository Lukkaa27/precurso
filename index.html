Comandos para git:

  1. Primer comando: git init (Para iniciar git, 1 sola vez por proyecto)
  2. Segundo comando: git status (Para checkear el estado de git en nuestro proyecto)
  3. Tercer comando: git add . (Para trackear todos los archivos)
  4. Cuarto comando: git commit -m "Mensajito" (Para hacer el paquetito de una versión, que podemos subir a la nube)
  5. git branch -M main
  6. git remote ...
  7. git push
